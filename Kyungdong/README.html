경동 사이트 클론코딩

HTML, CSS, JavaScript를 활용해 경동 공식 웹사이트의 메인 페이지를 클론코딩한 포트폴리오 프로젝트입니다.

📌 프로젝트 소개

본 프로젝트는 경동 공식 사이트의 레이아웃과 주요 UI를 참고하여 제작한 클론코딩 작업물입니다.

웹 페이지의 구조를 직접 분석하고, 메인 비주얼 영역과 슬라이드 인터랙션을 구현하며 실제 웹사이트와 유사한 사용자 경험을 제공하는 데 중점을 두었습니다.

  • 원본 사이트: http://www.kyungdong.co.kr/
  • 작업 유형: 웹사이트 클론코딩
  • 제작 목적: 퍼블리싱 실습 및 취업 포트폴리오
  • 기여도: 100%

🛠 사용 기술

구분 기술

Script JavaScript (ES6)
UI Swipe / Slide

🎞 메인 비주얼 슬라이드

  • 메인 비주얼 영역 슬라이드 구현
  • 이전 / 다음 버튼으로 슬라이드 이동
  • 자연스러운 화면 전환 효과
  • 사용자 인터랙션 기반 콘텐츠 변경

  • 이미지 및 텍스트 스타일링

  • 원본 사이트 분위기를 참고한 컬러 및 여백 구성

📖 배운 점

이번 프로젝트를 통해 실제 기업 웹사이트의 UI를 분석하고 이를 HTML, CSS, JavaScript로 구현하는 과정을 경험했습니다.

특히 메인 비주얼 슬라이드 기능을 구현하면서 DOM 제어와 이벤트 처리에 대한 이해를 높일 수 있었으며, 정적인 페이지를 넘어 사용자와 상호작용하는 웹 페이지 제작 경험을 쌓을 수 있었습니다.
